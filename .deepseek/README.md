# .deepseek - AI Local Memory

This directory stores persistent context about the **time-note** project for AI assistants. It helps maintain continuity across sessions without relying on conversation history.

## Structure

| Folder | Purpose |
|---|---|
| `project/` | Project overview, goals, tech stack, architecture |
| `components/` | Component inventory, props, state management |
| `decisions/` | Key design decisions and rationale |
| `known-issues/` | Bugs, quirks, edge cases |
| `workflows/` | Development workflows, commands, patterns |

## How to Use

1. Start each session by reading `README.md` files that are relevant
2. Update files when you learn new information about the project
3. Keep entries concise and actionable
4. Focus on what's non-obvious or frequently forgotten
